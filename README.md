# Security Monitoring System (Diploma)

ESP32-based security system with:
- door sensor
- motion sensor (PIR)
- tamper detection
- panic button
- MQTT communication

## Architecture
ESP32 → MQTT → Backend → Telegram bot

## Features
- Event collection
- JSON telemetry
- Alarm detection (planned)
- Event correlation (planned)

## Tech stack
- ESP32 (C++)
- MQTT (HiveMQ)
- Backend (planned)
- Telegram bot (planned)
